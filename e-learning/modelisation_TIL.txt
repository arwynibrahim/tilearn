═══════════════════════════════════════════════════════════════

5.1 SÉCURITÉ
┌─────────────────────────────────────────────────────────────┐
│ RG01 — Chiffrement TLS 1.3 obligatoire tous endpoints      │
│ RG02 — AES-256 pour données au repos (RDS, S3)             │
│ RG03 — Conformité RGPD : consentement, droit à l'oubli,    │
│        portabilité, DPO                                    │
│ RG04 — Hash bcrypt cost 12 pour mots de passe              │
│ RG05 — JWT RS256 (15min), refresh tokens avec rotation     │
│ RG06 — Rate limiting par IP + user, CAPTCHA inscription    │
│ RG07 — OWASP ZAP + Snyk dans CI/CD                         │
│ RG08 — Audit logs 12 mois pour actions sensibles           │
│ RG09 — Pen test externe annuel                             │
└─────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────┐
│ RG10 — LCP < 2s sur 4G (10-15 Mbps)                        │
│ RG11 — Bundle JS initial < 200 Ko                          │
│ RG12 — Images WebP progressives, lazy loading, BlurHash    │
│ RG13 — Offline-first via Service Workers (PWA)             │
│ RG14 — Support Chrome, Firefox, Safari, Edge (2 versions)  │
